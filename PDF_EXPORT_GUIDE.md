# PDF Export Feature Guide

## Overview
The Intelligent Quiz Application now includes a **PDF Report Export** feature that generates comprehensive quiz reports.

## How to Export PDF Reports

### Step-by-Step Instructions:

1. **Complete a Quiz**
   - Take and finish any quiz
   - You'll be automatically redirected to the Summary screen

2. **Review Your Summary**
   - Check your score, accuracy, and grade
   - Review the detailed question-by-question breakdown

3. **Click "Download PDF Report" Button**
   - Look for the orange button with ðŸ“¥ icon
   - Located in the bottom panel of the Summary screen

4. **Choose Save Location**
   - A file dialog will appear
   - Navigate to your desired save location
   - Default filename format: `Quiz_Report_Username_YYYYMMDD_HHMMSS.pdf`
   - You can change the filename if desired

5. **Save the Report**
   - Click "Save" button
   - A success message will confirm the export
   - The message shows the full file path

## PDF Report Contents

Your PDF report includes:

### Header Section
```
INTELLIGENT QUIZ APPLICATION - DETAILED REPORT
==================================================
```

### User Information
- Username
- Category (Java, AI, Networking, Custom)
- Difficulty Level (Easy, Medium, Hard)
- Date & Time of quiz completion

### Performance Summary
- **Score**: X / Y (e.g., 4 / 5)
- **Accuracy**: XX.XX%
- **Grade**: Letter grade with description
  - A+ (90-100%): Excellent
  - A (80-89%): Very Good
  - B (70-79%): Good
  - C (60-69%): Satisfactory
  - D (50-59%): Pass
  - F (<50%): Fail

### Detailed Question Analysis
For each question:
- Question number and status (CORRECT/INCORRECT)
- Full question text
- Your answer
- Correct answer (if you got it wrong)

### Footer
- End of report marker
- Generation attribution

## File Format Notes

### Current Implementation
- **Format**: Formatted text file with .pdf extension
- **Readable**: Can be opened in any text editor
- **Compatible**: Works on all systems without external libraries

### For True PDF Format
To generate actual PDF files with formatting, fonts, and graphics:

1. **Download iText Library**
   - Visit: https://github.com/itext/itext7
   - Or use Apache PDFBox: https://pdfbox.apache.org/

2. **Add to Classpath**
   ```bash
   # Compile with library
   javac -cp .;itext7.jar IntelligentQuizApp.java
   
   # Run with library
   java -cp .;itext7.jar IntelligentQuizApp
   ```

3. **Benefits of PDF Libraries**
   - Proper PDF formatting
   - Custom fonts and colors
   - Tables and charts
   - Images and logos
   - Password protection
   - Digital signatures

## Common Issues & Solutions

### Issue: Button is disabled
**Solution**: Complete a quiz first. The button only activates after quiz completion.

### Issue: "No quiz result available to export"
**Solution**: The summary must be from a recently completed quiz. Take a new quiz.

### Issue: File not saving
**Solution**: 
- Check write permissions on selected directory
- Ensure sufficient disk space
- Try saving to a different location (e.g., Desktop)

### Issue: Cannot open PDF
**Solution**: 
- The current implementation creates formatted text files
- Open with any text editor (Notepad, Notepad++, VS Code)
- For true PDF, add iText library (see above)
